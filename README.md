# Retail Product Pricing Analysis Dashboard

## Overview

This project presents a retail pricing analysis dashboard built using **Power BI**. The dashboard analyzes product pricing trends across two global fashion brands: Zara and H&M. The objective is to compare pricing strategies, identify premium products, and visualize key retail insights.

---

## Project Objectives

* Compare average pricing between brands
* Identify the most expensive products
* Analyze product distribution across brands
* Build an interactive retail analytics dashboard
* Demonstrate data visualization and DAX skills

---

## Dataset

The dataset contains product-level information collected for fashion retail analysis.

### Columns

* **BRAND** – Brand name (Zara or H&M)
* **NAME** – Product name
* **DESCRIPTION** – Product description and details
* **PRICE** – Product price

### Dataset Summary

* Total Products: **9,929**
* Brands Analyzed: **Zara, H&M**

---

## Tools & Technologies

* **Microsoft Power BI** – Dashboard development
* **DAX (Data Analysis Expressions)** – Calculated measures
* **Power Query** – Data transformation and cleaning

---

## Key Metrics

The following KPIs were created using DAX:

* **Average Price**
* **Maximum Price**
* **Total Price**
* **Total Products**

These metrics provide a quick overview of the retail pricing structure.

---

## Dashboard Features

### KPI Cards

The dashboard highlights key metrics:

* Average Price
* Maximum Product Price
* Total Price Value
* Total Number of Products

### Average Price by Brand

A bar chart comparing the average product price between Zara and H&M.

### Top Expensive Products

A table displaying the highest-priced products in the dataset.

### Interactive Filters

Users can explore the dashboard using:

* Brand filter
* Product name filter

---

## Key Insights

* Zara products generally have a **higher average price** compared to H&M.
* Premium items such as leather jackets and boots appear among the most expensive products.
* H&M provides more **affordable pricing** compared to Zara.
